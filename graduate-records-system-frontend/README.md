
# Frontend - Graduate Records System

This is the React.js frontend of the Graduate Records System.

## Requirements

- Node.js (v18+ recommended)
- npm or yarn

## Installation

1. Open terminal and navigate to the frontend folder:
```bash
cd frontend
Install dependencies:

npm install

Configuration

If needed, create a .env file in the frontend folder:

REACT_APP_API_URL=http://localhost:5000

Running the Frontend
npm start


The React app will open in your browser at http://localhost:3000.
